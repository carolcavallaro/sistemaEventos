Início do programa

1. Mostrar eventos cadastrados
2. Mostrar cadastro de eventos

1.1. Ao abrir evento cadastrado mostrar:
1.1.1. Mostrar nome do palestrante
1.1.2. Mostrar data do evento
1.1.3. Mostrar quantidade de participantes
1.1.3.1. Listar participantes cadastrados
1.1.3. Mostrar cadastro de participantes
1.1.3.1. Se houver até 99 participantes cadastrados, permitir nova inscrição
1.1.3.2. Se tiver 100 inscritos, informar que evento excedeu o limite de participantes permitido

2. Cadastro de eventos
2.1. Receber nome do palestrante
2.2. Receber data do evento
2.2.1. Se a data for posterior a data atual, permitir cadastro do evento
2.2.2. Senão, alertar data inválida

3. Cadastro de participantes
3.1. Receber nome do participante
3.2. Receber data de nascimento do participante
3.2.1. Se tiver 18 anos ou mais, permitir cadastro
3.2.2. Se tiver menos de 18 anos, alertar que não é permitido o cadastro
